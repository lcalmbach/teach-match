# GitLab Duo Workflow Agent Configuration - Teach-Match

## Project Overview

**Teach-Match** is a Django-based teacher substitution management system for schools. It manages departments, teachers, classes, subjects, and substitution assignments by importing data from the Untis scheduling system.

### Purpose
- Manage teacher availability for substitutions
- Import and sync data from Untis XML/CSV exports
- Match available teachers with substitution needs
- Track teacher qualifications and subjects

## Technology Stack

### Backend
- **Python**: 3.11+
- **Django**: 5.01+
- **Database**: SQLite (development), PostgreSQL (production)

### Key Dependencies
- Django REST Framework (if using API)
- Pillow (for profile pictures)
- chardet (for encoding detection)

### External Systems
- **Untis**: School scheduling system (data source)
